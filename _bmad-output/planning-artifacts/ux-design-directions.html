Evaluation Guide
Use these criteria when comparing directions. Pick a favorite or mix elements across directions.
Customer Menu — What to look for

First impression

Which feels most like a restaurant menu, not a software UI?

Food photos

D1 and D3 use images. D2 is text-only. Which matches your restaurant partners' content?

Scan speed

Which lets Aisha find "Spaghetti Carbonara" fastest without reading every word?

Trust

Which feels most like a place you'd comfortably place a food order?

Admin Order Feed — What to look for

Glanceability

Which can Marco read at arm's length while walking the floor?

Load capacity

Which holds up better when 8–10 orders arrive simultaneously?

Action speed

Direction A has inline buttons. Direction B has compact text links. Which feels faster to act on?

Calm

Which feels more like a reliable kitchen window and less like an alarming inbox?

Decisions to make
1. Customer menu: D1 (images + rows), D2 (text-only), or D3 (image cards)?
2. Admin feed: Direction A (full cards) or Direction B (compact list)?
3. Confirmation screen: any copy or layout changes?
4. Desktop admin: does the 2-column split feel right?
